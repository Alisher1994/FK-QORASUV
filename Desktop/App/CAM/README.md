# Демонстрация камеры

1. Убедитесь, что у вас установлен Python 3.
2. В терминале (`powershell.exe`) перейдите в папку проекта и запустите HTTP-сервер:
   ```powershell
   cd c:\Users\LOQ\Desktop\App\CAM
   python -m http.server 8000
   ```
3. Откройте браузер (Chromium-браузеры, Edge, Firefox) и перейдите по адресу `http://localhost:8000`.
4. Нажмите на кнопку «Запустить камеру» и разрешите доступ к встроенной камере.
5. Если браузер не запрашивает разрешение, проверьте блокировки камеры и используйте HTTPS или `localhost`.
